# @stewardtrack/mobile

StewardTrack mobile application for iOS and Android, built using Capacitor.

## Prerequisites

Before setting up the mobile app, ensure you have:

- **For iOS:**
  - macOS with Xcode 15+
  - CocoaPods installed (`sudo gem install cocoapods`)
  - Apple Developer account (for device testing and App Store)

- **For Android:**
  - Android Studio with SDK 34+
  - Java 17+
  - Android device or emulator

## Setup (Phase 3)

The Capacitor integration will be completed in Phase 3. Once set up:

1. **Install dependencies:**
   ```bash
   pnpm install
   ```

2. **Build the web app:**
   ```bash
   pnpm --filter @stewardtrack/web build:static
   ```

3. **Sync with native projects:**
   ```bash
   pnpm sync
   ```

4. **Open in IDE:**
   ```bash
   # iOS
   pnpm ios

   # Android
   pnpm android
   ```

5. **Run on device/simulator:**
   ```bash
   # iOS
   pnpm run:ios

   # Android
   pnpm run:android
   ```

## Configuration

The `capacitor.config.ts` file (created in Phase 3) will contain:

- App ID: `com.stewardtrack.app`
- App Name: `StewardTrack`
- Web directory: `../web/out` (Next.js static export)
- Plugin configurations for push notifications, camera, biometrics, etc.

## Native Plugins (Phase 3-4)

The following Capacitor plugins will be added:

- `@capacitor/app` - App lifecycle
- `@capacitor/camera` - Photo capture
- `@capacitor/device` - Device info
- `@capacitor/haptics` - Tactile feedback
- `@capacitor/keyboard` - Keyboard handling
- `@capacitor/preferences` - Secure storage
- `@capacitor/push-notifications` - Push notifications
- `@capacitor/splash-screen` - Launch screen
- `@capacitor/status-bar` - Status bar customization
- `@capacitor-community/biometric-auth` - Face ID / Fingerprint
- `@capacitor-community/safe-area` - Notch handling

## Development Workflow

1. Make changes to the web app (`apps/web/`)
2. Build with `CAPACITOR_BUILD=true` for static export
3. Sync to native projects with `cap sync`
4. Test on simulator/device
